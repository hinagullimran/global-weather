# 🌍 Global Weather - Precision Forecasts

A modern, high-performance web application that provides real-time weather data for any city or country in the world. Built with a focus on **rich aesthetics**, **glassmorphism**, and **premium user experience**.

![Preview](https://via.placeholder.com/800x400/1a2a6c/ffffff?text=Global+Weather+App) *(Note: Placeholder image for layout purposes)*

## ✨ Features

- **🌐 Global Reach**: Search for weather data by City or Country name.
- **🚩 National Flags**: Automatically displays the national flag of the selected location.
- **🌦️ Dynamic UI**: The background gracefully transitions based on current weather conditions (e.g., sunny, rainy, snowy).
- **💎 Glassmorphic Design**: A premium interface featuring backdrop-blur, smooth transitions, and vibrant gradients.
- **⚡ High Performance**: Uses the **Open-Meteo API** (no API key required) for lightning-fast data fetching.
- **🔍 Smart Search**: Intelligent autocompletion with `datalist` dropdowns for popular global hubs.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure and modern form elements.
- **CSS3**: Advanced layouts, glassmorphism, and dynamic animations.
- **JavaScript (ES6+)**: Fetch API, Geocoding, and real-time DOM updates.
- **APIs**:
  - [Open-Meteo](https://open-meteo.com/) (Weather & Geocoding)
  - [FlagCDN](https://flagcdn.com/) (National Flags)

## 🚀 How to Run Locally

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Start exploring!
